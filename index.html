Galactic Bounce

A physics-based arcade shooting game

Project for COMS W4167: Physically Based Computer Animation. This game is a fast-paced 2D arcade shooter with realistic gravity forces and collision detection/response. Playable on Windows, Mac, and Linux, and allows players to create custom levels! Created with Java and Slick2D, an OpenGL/OpenAL library for Java game development.

yelp!help

A friendly alternative to Yelp.com's bookmarking system

Final project for COMS W4170: User Interface Design. Created by myself, Shannon Lee, Siddhi Mittal, and Stephen Somuah, yelp!help is designed to be an easier way to create, search, and filter bookmarks from Yelp. It implements several convenient features that Yelp's bookmarking system lacks, such as parallel filters, additional sort options, and batch select/delete.

RoboPathFinder

A pathfinding system for the iRobot Create

Project for COMS W4733: Computational Aspects of Robotics. This program takes as input sets of coordinates to define the map, obstacles and start/goal points for the robot, and computes and visually displays the shortest path to the goal. Utilizes various algorithms in tandem, such as a Graham scan for growing obstacles, visibility graph construction, and Dijkstra's algorithm/A* search. Path is written out as a sequence of commands to be fed to the robot.

TeapotSun

Fully explorable random solar system generator

Final project for COMS W4160: Computer Graphics. Created in C++ using OpenGL and glut, this simple application generates a random solar system with a sun, 8 planets, and an asteroid belt. Each planet and asteroid is generated at a random distance and has its own size, texture, rotation speed and revolution speed. The system is explorable via first-person shooter style controls, and has keyboard shortcuts for changing speed, toggling orbit paths, and generating a new system.

TeapotSun on GitHub

A simple but useful Pokemon web app

Project for COMS W4111: Intro to Databases. AI Games

Simple games with human and AI players

Projects for COMS W4701: Artificial Intelligence. Games include: (1) Gomoku, a strategy board game similar to Tic-Tac-Toe except can be played on any size grid for any amount in a row. Has several modes, including human vs human and human vs AI. The AI player uses an alpha-beta adversarial search algorithm to determine moves. (2) Sokoban, a popular tile based puzzle game that involves pushing boxes into goals. Contains an interactive mode that allows humans to play, as well as an AI mode in which a solver uses various search algorithms (BFS, DFS, UCS, A*) to determine the shortest winning move sequence.

A gesture-based image browser

Another project for COMS W4170: User Interface Design. This web app uses a modified version of the $1 unistroke gesture recognizer to browse images of cats. The user simply clicks and drags the mouse over the images and performs any of the available gestures.

Thrown Controllers

Panel-based game show for video game trivia

An adaptation of the panel-based game show done by The Runaway Guys at PAX East 2013, Thrown Controllers is a game where two teams compete in a series of video game trivia challenges and single/multiplayer gaming challenges to get the highest score. It is panel based, in a similar style to Jeopardy. This desktop app was created with Java and Swing as an activity for students at iD Tech Camps during my time as an instructor. Best used in conjunction with a large screen or a projector!

Music & YouTube How I spend time on the Internet.

Music is one of my biggest hobbies, both listening and creating. I started playing the piano almost 12 years ago, and have picked up a half-dozen or so instruments along the way. I am a fan of all genres and love to experiment. Nowadays I am spending much of my free time doing music production, both for myself and others. All of my music is free to stream and download on my Bandcamp page.

Video is just as big a hobby for me as music, so I wanted to figure out a meaningful way to combine the two. I eventually discovered videosongs: music videos with multiple camera angles and fast-paced edits that show off the entire live recording process. Essentially, a videosong is a is a type of medium with two rules:
1. What you see is what you hear (no lip-syncing for instruments or voice).
2. If you hear it, at some point you see it (no hidden sounds).

I create videosongs under the moniker goldinsounds, and most of my videos can be found on my YouTube channel of the same name. I produce videos for other channels as well, and can be seen at places like here and here. My most popular YouTube music endeavor to date is Video Game Cover Week, a two-month project that I completed and released in February 2013. I recorded and produced an EP consisting of 7 songs from popular video games, made a videosong for each, and released one video every day during the week of February 3. Since then, the project has ammassed over 30,000 views on YouTube, over 2000 downloads on Bandcamp, and has been featured on FUSE and the OFFICIAL Tetris Facebook page!
